# Flutter 项目管理器 - 需求文档

## 背景
- 用户有 3 个 Flutter 项目：`links2-control-mobile`、`links2-control-panel`、`links2-power-mobile`
- 之前用 VSCode 是因为它的 Flutter 扩展提供了快捷操作面板（运行、热重载等）
- 现在用 Claude Code 开发，不需要打开 VSCode
- **需要一个独立的工具提供同样的快捷操作功能**

## 核心需求

### 1. 应用类型
**Flutter Desktop (macOS 应用)**
- 用户选择用 Flutter 来开发这个工具
- 可以复用 Flutter 开发经验

### 2. 核心功能
**快捷操作按钮**（必需）
- ▶️ 运行 (flutter run -d <device>)
- 🔄 热重载 (快捷键/按钮触发，发送 'r' 到进程)
- 🔃 热重启 (发送 'R' 到进程)
- ⏹️ 停止 (发送 'q' 到进程)

### 3. 使用场景
```
开发者工作流程:
1. 打开 "Flutter 项目管理器"
2. 选择要操作的项目
3. 选择目标设备（iPhone/模拟器/macOS）
4. 点击 "运行" 按钮
5. 代码修改后，点击 "热重载" 按钮
6. 完成后点击 "停止"
```
